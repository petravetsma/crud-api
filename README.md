# CRUD API

 Simple CRUD API with in-memory database underneath.

 ### Prerequisites:
- Install Node.js (version 22.xx.xx)
- Install all dependencies use `npm install`
- Create file .env (use .env.template as a basis)
- Set port in .env file (example, PORT=3002)

 ### Commands:

Run Node.js CRUD server DEV mode

`npm run start:dev`

Run Node.js CRUD server PROD mode

`npm run start:prod`

Run tests

`npm run test`

---

Use app like [postman](https://www.postman.com/) to test app.
